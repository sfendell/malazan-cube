**→ [Browse the cube](https://sfendell.github.io/malazan-cube/)**

## Developer needs

- **Python 3**
- **Magic Set Editor**: https://magicseteditor.boards.net/page/downloads
- **M15 Magic Pack**: https://github.com/MagicSetEditorPacks/M15-Magic-Pack

---
## How to edit cards:
* Open mse-set with the mse editor
* Make whatever changes you feel like
* Run finalize.py
* Make PR with all changed files (there should be changes to cards.json, exported_cards, and text, at least)
